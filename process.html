The Structure of V-Model

The Structure


•

Requirement analysis - The requirements of the project is analysed and documented .

•

System design - The system design is created ( includes architecture and detailed specifications.)

•

Architectural design - The data transfer and communication between the internal modules and other systems.

•

Module design - The detailed design of each software module is developed.

•

Coding/ implementation - The software is developed and tested at the module level.
